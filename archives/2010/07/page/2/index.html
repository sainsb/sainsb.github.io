Friday wrap-up

The small things:

  • Created a robots.txt file for our external web server in hopes that it cuts down on the spider bandwidth.
  • Tried to get the external app web services to be SOAP compliant through the use of XMLInclude attribute tags.  May need to parse search type client side and break out parseText. Update - I never did get this working due to namespace collisions with my webservices. Since I was consuming them in a REST/JSON fashion, it was never a problem, but SOAP is much stricter.
  • Working on map annotation for users.  Got the API working but worried about print service….will need to detect them clientside and send the point and text to printserver.asmx…
  • Tiled Aerials Express out to level 17.
  • Learned that you can have 2 remote sessions open if you reconnect and the terminal isn’t happy.
  • Learned that memory usage in the task manager doesn’t nec. indicate CPU usage.
  • Listening to: Disco Biscuits live at Bonnaroo
